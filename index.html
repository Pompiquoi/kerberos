Welcome to the Kerberos Project

This project demonstrates the setup and configuration of a Kerberos authentication system, focusing on enabling seamless integration between a Linux Kerberos server and a Windows client. The aim is to achieve secure, ticket-based authentication in a cross-platform environment.

Project Objectives

  • Install and configure an MIT Kerberos server on a Linux machine.
  • Integrate a Windows client with the Kerberos server to enable secure authentication.
  • Provide an intuitive presentation of Kerberos technology, its setup, and practical use cases.

Technologies and Tools

  • Kerberos: network authentication protocol based on tickets issued by a centralized server called the Key Distribution Center (KDC).
  • Group Policy Editor: A Windows administration tool to configure password requirements, and define what applications or settings users can change.
  • MIT Kerberos Ticket Manager: A GUI tool on Windows to manage and inspect Kerberos tickets.
